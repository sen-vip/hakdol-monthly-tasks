# 학돌 월별 업무판 🗓️

학교 행정실의 월별 업무 흐름을 캘린더와 업무판으로 한눈에 확인하는 웹앱입니다.
“우리 학교 이번 달에 뭐 하지?”라는 질문에서 시작해, 이번 달 행정업무·학사일정·직접 추가 일정을 함께 확인할 수 있도록 구성했습니다.

---

## ✨ 주요 기능

### 1. 이번 달 업무판

현재 월 기준으로 행정실 업무를 자동 정리합니다.

* 이번 달 전체 업무 수 확인
* 제출업무 / 챙길업무 구분
* 완료율 확인
* 월초·중순·월말 등 시기별 업무 확인

---

### 2. 월간 캘린더

이번 달 캘린더에서 날짜별 업무 흐름을 확인할 수 있습니다.

* 행정업무 요약 표시
* 학사일정 표시
* 직접 추가 일정 표시
* 날짜 클릭 시 해당 날짜의 상세 업무 확인

캘린더에는 긴 업무명을 모두 넣지 않고, 날짜별 요약을 먼저 보여줍니다.
상세 내용은 날짜를 클릭하면 아래 패널에서 확인할 수 있습니다.

---

### 3. 학교 설정

학교를 선택하면 해당 학교 기준으로 사용할 수 있습니다.

* 시도교육청 선택
* 학교명 검색
* 학교 정보 저장
* 선택한 학교명 상단 표시

학교 정보와 직접 추가 일정은 현재 브라우저에 저장됩니다.

---

### 4. 업무 보기 필터

필요한 업무만 골라볼 수 있습니다.

* 전체업무
* 챙길업무
* 제출업무
* 업무분장별 필터

  * 예산
  * 급여
  * 세입
  * 지출
  * 계약
  * 물품재산
  * 환경위생
  * 기록물
  * 민방위
  * 안전보건
  * 인사
  * 직원교육 등

검색창을 통해 업무명, 분류, 근거, 안내부서도 검색할 수 있습니다.

---

### 5. 완료 / 해당없음 관리

업무별 상태를 관리할 수 있습니다.

* **완료**: 처리한 업무로 표시
* **해당없음**: 우리 학교에는 해당하지 않는 업무로 표시
* **미완료**: 아직 처리하지 않은 기본 상태

완료한 업무는 줄긋기와 완료톤으로 표시됩니다.
해당없음 업무는 회색톤으로 표시되며 완료율 계산에서 제외됩니다.

---

### 6. 우리 학교 일정 추가

학교별로 따로 챙겨야 하는 일정을 직접 추가할 수 있습니다.

입력 항목:

* 날짜
* 일정명
* 분류

  * 학사
  * 행정
  * 메모
* 메모

추가한 일정은 캘린더와 날짜 상세 패널에 표시됩니다.
직접 추가 일정은 브라우저에 저장되며, 업무 통계나 완료율에는 포함되지 않습니다.

---

## 🧭 사용 흐름

1. 웹앱에 접속합니다.
2. 현재 월 업무판이 자동으로 표시됩니다.
3. 학교를 검색하고 선택합니다.
4. 이번 달 캘린더에서 날짜별 업무를 확인합니다.
5. 날짜를 클릭해 상세 업무를 봅니다.
6. 필요한 경우 업무분장 필터로 내 업무만 골라봅니다.
7. 처리한 업무는 완료로 표시합니다.
8. 우리 학교에 해당 없는 업무는 해당없음으로 표시합니다.
9. 학교별 추가 일정은 `+ 일정 추가` 버튼으로 입력합니다.

## 🗂️ 데이터 기준

월별 업무 데이터는 정리된 엑셀 파일을 기준으로 구성했습니다.

주요 항목:

* 월
* 대략적 일정
* 업무분류
* 주요 업무명
* 안내부서
* 근거 법령
* 간략내용
* 비고

검증단 의견 등 참고용 정보가 있는 경우 상세보기에서 확인할 수 있도록 구성했습니다.

---

## 💾 저장 방식

이 웹앱은 사용자의 브라우저 저장소를 활용합니다.

저장되는 정보:

* 선택한 학교 정보
* 업무 완료 상태
* 해당없음 상태
* 직접 추가한 우리 학교 일정

서버로 개인정보나 입력 데이터가 자동 전송되지 않습니다.
