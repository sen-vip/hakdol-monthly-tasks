# 📅 학돌 월별필수업무

학교 행정실 담당자가 한 해 동안 처리해야 하는 주요 업무를 월별로 확인하고, 완료 여부와 메모를 저장하며, 다가오는 업무를 미리 준비할 수 있도록 돕는 업무지도형 웹앱입니다.

## 주요 기능

- 월별 필수업무 보기
- 연간 업무 조망
- 이번 달 업무 자동 표시
- 검색 및 업무분류 필터
- 업무 상세보기
- 업무분류별 컬러 딱지
- 비로그인 공개 열람
- 로그인 시 완료 체크 저장
- 중요 표시
- 해당없음 처리
- 개인 메모 저장
- 내 추가업무 등록
- 프린트/PDF용 보기

## 데이터 기준

기본 업무 데이터는 `학돌_월별필수업무 0620정리본.xlsx`의 내용을 바탕으로 구성했습니다.

앱에 포함한 항목은 다음과 같습니다.

- 월
- 대략적 일정
- 업무분류
- 주요 업무명
- 안내부서
- 근거 법령
- 간략내용
- 비고

`검증단 의견` 열은 내부 검토 의견이므로 앱 데이터에서 제외했습니다.

## 저장 방식

기본 업무 목록은 공개 데이터로 제공됩니다.
로그인한 사용자의 완료 체크, 중요 표시, 해당없음, 메모, 추가업무만 계정별로 저장됩니다.

서버 저장은 Supabase Auth와 Supabase DB를 사용합니다.

## Supabase 설정 방법

1. Supabase 프로젝트를 생성합니다.
2. Supabase SQL Editor에서 `supabase-schema.sql` 내용을 실행합니다.
3. Project Settings → API에서 Project URL과 anon public key를 확인합니다.
4. `supabase-config.js`에 값을 입력합니다.

```js
window.HAKDOL_SUPABASE_CONFIG = {
  url: "https://YOUR_PROJECT.supabase.co",
  anonKey: "YOUR_ANON_KEY"
};
```

5. 정적 웹호스팅에 배포합니다.

## 민감정보 입력 금지

이 앱은 월별 업무 체크 및 개인 메모 저장용입니다.
학생·교직원 개인정보, 급여 세부자료, 계좌번호, 민원 내용 등 민감정보는 입력하지 마세요.
