# 📅 학돌 월별필수업무

학교 행정실에서 매월 반복되는 필수 업무를 한눈에 확인하고 관리할 수 있는  
**월별 업무 체크 보드**입니다.

👉 **바로가기**  
https://sen-vip.github.io/hakdol-monthly-tasks/

---

## ✨ 이런 용도로 만들었어요

행정실 업무는 매월 반복되지만,  
막상 담당자가 바뀌거나 바쁜 시기가 되면  
“이번 달에 꼭 해야 하는 업무가 뭐였지?” 하고 놓치기 쉽습니다.

**학돌 월별필수업무**는 월별 필수 업무를 정리하고,  
업무 설명·담당 메모·체크 상태를 함께 관리할 수 있도록 만든 도구입니다.

---

## 주요 기능

### 1. 월별 필수업무 확인

1월부터 12월까지 월별로 필요한 업무를 확인할 수 있습니다.

- 월별 업무 목록 확인
- 업무 제목, 설명, 담당 메모 관리
- 행정실 반복 업무 중심 구성

---

### 2. 업무 체크 기능

완료한 업무는 체크할 수 있습니다.

- 체크 상태는 이 PC에 저장
- 새로고침 후에도 체크 상태 유지
- 월별 업무 진행 상황 확인 가능

---

### 3. 검색 및 필터

필요한 업무를 빠르게 찾을 수 있습니다.

- 업무명 검색
- 설명 내용 검색
- 월별 필터
- 분류별 확인

---

### 4. 글쓰기 블록

업무별로 필요한 내용을 직접 적을 수 있습니다.

예시:

- 우리 학교 처리 방식
- 담당자 참고사항
- 제출 기한
- 관련 시스템
- 주의사항
- 다음 담당자에게 남길 말

---

### 5. 엑셀 / JSON 관리

업무 데이터를 백업하거나 다시 불러올 수 있습니다.

- 기본 업무 데이터 제공
- JSON 백업
- JSON 불러오기
- 엑셀 기반 업무자료와 연계 가능

---

## 저장 방식

이 도구는 입력한 내용을 외부 서버로 보내지 않습니다.

체크 상태와 메모는 사용자의 브라우저에 저장됩니다.

즉,

- 같은 PC
- 같은 브라우저
- 같은 주소

에서 다시 열면 저장된 내용을 계속 사용할 수 있습니다.

다만 브라우저 캐시 삭제, 다른 PC 사용, 시크릿 모드 사용 시 저장 내용이 보이지 않을 수 있으므로  
중요한 내용은 JSON 백업을 권장합니다.

---

## 사용 방법

1. 아래 링크로 접속합니다.

   https://sen-vip.github.io/hakdol-monthly-tasks/

2. 현재 월 또는 확인하고 싶은 월을 선택합니다.

3. 업무 목록을 확인합니다.

4. 완료한 업무는 체크합니다.

5. 필요한 경우 글쓰기 블록에 학교별 메모를 남깁니다.

6. 중요한 데이터는 JSON 백업으로 저장합니다.

## 추천 활용 방식

### 신규 담당자용

월별 필수업무를 보면서  
“이번 달에 무엇을 해야 하는지” 확인하는 용도로 사용할 수 있습니다.

### 기존 담당자용

반복 업무를 체크하고,  
작년 또는 전임자의 메모를 참고하는 업무 보드로 사용할 수 있습니다.

### 행정실 공용 업무판

행정실 PC에서 함께 열어두고  
월별 업무 진행 상황을 공유하는 용도로 사용할 수 있습니다.

## 주의사항

이 도구는 행정업무를 보조하기 위한 참고용 도구입니다.

학교별 업무 기준, 교육청 지침, 내부 결재 절차, 에듀파인 처리 방식은 다를 수 있으므로  
최종 업무 처리는 반드시 소속 기관의 기준에 따라 확인해야 합니다.

특히 다음 사항은 담당자가 직접 확인해야 합니다.

- 교육청 공문
- 학교 내부 결재 기준
- 예산 집행 기준
- 계약 및 지출 관련 규정
- 업무별 제출 기한
- 학교별 담당자 분장
