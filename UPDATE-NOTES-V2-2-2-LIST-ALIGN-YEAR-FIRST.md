# v2.2.2 목록 폭 정렬 및 연간보기 우선 업데이트

## 반영 내용
- `이번 달 챙길 업무` 행을 표처럼 정렬했습니다.
  - 체크박스 / 날짜 / 업무딱지 / 업무명 칸의 폭을 고정
  - 업무명과 설명 시작 위치가 행마다 흔들리지 않도록 보정
- 기본 월 보기에서 아래 `월별 필수업무` 중복 목록을 숨겼습니다.
- `연간보기` 버튼을 누르면 연간 업무 조망 카드가 화면 상단 흐름으로 올라오도록 조정했습니다.
- 연간보기 상태에서는 `이번 달 챙길 업무`와 중복 월별 목록이 보이지 않도록 정리했습니다.
- 기존 로그인, 가입완료 페이지, 제출성 업무 제외, 업무딱지 컬러, 학교 캘린더/나이스 연동 기능은 유지했습니다.
