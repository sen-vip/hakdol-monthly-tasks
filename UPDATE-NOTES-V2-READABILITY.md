# v2.0 업무판 가독성 개선 업데이트

## 반영 내용
- 상단 영역을 더 짧고 단단하게 정리했습니다.
- 로그인 안내 문구를 작은 보조 문구로 낮췄습니다.
- 업무분장 선택을 월 선택보다 먼저 배치했습니다.
- “언제 해야 하나요?” 월 흐름 필터를 업무분장 옆에 배치했습니다.
- 상세 필터 버튼을 기본 노출하고, 세부 필터 영역은 접힌 상태를 유지했습니다.
- 이번 달 요약 숫자칩을 더 작고 선명하게 정리했습니다.
- 전체 컬러를 연한 회청색 배경 + 선명한 남청/블루 포인트로 조정했습니다.
- 모바일에서는 월 선택 패널을 숨겨 상단 길이를 줄였습니다.

## 유지한 내용
- 업무 카드 본문 내용은 숨기거나 줄이지 않았습니다.
- 자세히 보기 접힘 구조는 새로 만들지 않았습니다.
- 기존 검색, 필터, 완료 체크, 로그인 저장 로직은 유지했습니다.
