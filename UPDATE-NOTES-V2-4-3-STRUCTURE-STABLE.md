# 2.4.3 구조 안정화 업데이트

- 상단 업무판/우리학교 영역 간격과 시각 위계를 정리했습니다.
- `달력`, `통합달력` 반복 표현을 줄이고 `우리 학교 일정`, `6월 한눈에` 중심으로 명칭을 정리했습니다.
- 브라우저 기본 날짜 선택창 대신 1~12월 격자형 커스텀 월 선택 패널을 추가했습니다.
- 달력 상단 배지 줄과 설명 문구를 분리해 겹침 오류를 수정했습니다.
- 챙김업무는 더 눈에 띄는 민트/초록 계열로, 제출업무는 차분한 살구/코랄 계열로 조정했습니다.
- 업무딱지 색상과 둥글기, 업무 목록 폭 기준을 정리했습니다.
- `neis-extension.js`의 중복 const 선언 오류를 수정했습니다.
