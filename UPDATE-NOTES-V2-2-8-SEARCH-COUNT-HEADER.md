# v2.2.8 검색/카운트 버그 수정 + 상단 헤더 정리

## 반영 내용

- 검색 대상 필드를 업무명, 업무분장, 시기, 설명, 근거, 안내부서, 비고, 출처까지 통합했습니다.
- 카운트 계산 기준과 목록 필터 기준을 분리하고, 같은 시기 그룹 함수를 사용하도록 정리했습니다.
- 월초/중순/월말 요약 칩 숫자와 실제 목록 결과가 다르게 나오는 문제를 보정했습니다.
- 요약 칩의 `전체`를 눌러도 현재 보기 상태(전체/챙길 업무/제출 업무)는 유지하고 시기/상태 필터만 초기화되도록 조정했습니다.
- 업무분장/필터 조작 후 숫자 배지가 사라지지 않도록 카운트 계산을 안정화했습니다.
- 상단 업무판 헤더의 왼쪽 파란 세로선과 큰 배너 느낌을 줄이고, 낮은 업무판 헤더 스타일로 정리했습니다.
- 필터 버튼과 숫자 배지의 둥근 정도와 강조감을 낮춰 전체 톤과 맞췄습니다.

## 유지한 기능

- 업무딱지 컬러
- 제출성 업무 판단 로직
- 메모 간소화
- 상세 보기
- 완료 체크
- 로그인/가입완료 페이지
- 학교 일정 확인 및 통합달력 접힘 구조
