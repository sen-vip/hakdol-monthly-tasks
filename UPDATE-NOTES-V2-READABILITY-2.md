# v2.0.1 이번 달 챙길 업무 가독성 보정

## 반영 내용
- 나이스 확장 영역의 `이번 달 챙길 업무` 카드 가독성 개선
  - 제목 글자 크기와 굵기 강화
  - 카드 내부 여백 정리
  - 기간/분류 딱지 묶음 정돈
  - 반복 안내 문구 제거
  - 모바일에서는 1열로 자연스럽게 전환
- 제출성 업무 제외 기준 보강
  - 안내부서가 있고 제목이 `~조사`, `~신청`, `~제출`로 끝나는 업무를 제출업무로 판단
  - 기존 `~협조` 기준도 유지
- `이번 달 챙길 업무` 추천 목록에서도 제출성 업무 제외

## 유지한 내용
- 기존 데이터 구조 유지
- 기존 필터/검색/로그인/완료 체크 로직 유지
- 업무 카드 본문 접기 구조 추가하지 않음
