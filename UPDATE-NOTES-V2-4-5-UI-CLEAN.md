# v2.4.5 UI 정리 업데이트

## 핵심 변경
- 상단바를 앱명·학교명·현재월·도움말 중심으로 재정리했습니다.
- 학교 정보는 학교명을 왼쪽에 크게, 교육청/학교코드/건수/업데이트 정보는 보조 정보로 분리했습니다.
- 직접추가는 모든 영역에서 회색/슬레이트 계열로 통일했습니다.
- 제출/챙김은 행동 배지로 유지하고, 업무분류 딱지는 저채도 보조 라벨로 낮췄습니다.
- 별표(중요 표시)는 비로그인 상태에서도 localStorage에 저장되도록 로그인 차단을 제거했습니다.
- 학교 설정 패널의 하단 모서리 깨짐, 작은 글씨, 입력창/버튼 높이를 개선했습니다.
- 달력 날짜 클릭 시 과한 파란 선택 박스 느낌을 줄였습니다.
- 캐시 방지 버전값을 v245-ui-clean으로 변경했습니다.
