# v2.2.3 기본보기/필터 흐름 정리

- 기본 진입값을 `전체보기`로 변경했습니다.
- 이번 달 업무 목록 정렬을 월초 → 월중/중순 → 월말 흐름으로 보정했습니다.
- 필터 흐름을 `월 선택 → 어떤 업무분장인가요?`로 단순화하고, `언제 해야 하나요?` 필터는 숨겼습니다.
- 필터 버튼을 진한 남색 덩어리 대신 연한 블루 선택 상태로 정리했습니다.
- 목록의 메모 입력은 바로 로그인 팝업을 띄우지 않고, 상세 화면 안에서 확인/저장하도록 정리했습니다.
- 학교 캘린더 영역에서는 월별업무 중복 연결을 제거하고, 통합달력과 학사일정 연계 추천업무만 보여주도록 정리했습니다.
