# 2.4.5-fix3 업데이트 노트

## 목적
달력 카드의 주인공을 학교명이 아닌 현재 월로 다시 세우고, 검색/보기 필터 겹침과 업무분류 딱지 색상 체계를 정리했습니다.

## 반영 내용
- 달력 카드 상단의 학교명/교육청/학교코드 정보 반복 표시 제거
- 달력 카드 상단을 월 컨트롤 중심 구조로 재정렬
- 보조 정보 문구를 `학사 124 · 시간표 9 · 직접 0 · 업데이트 22:15` 형태로 축소
- 월 컨트롤을 카드 상단 가까이 배치
- 검색창과 보기 필터 버튼 줄 분리
- `전체/챙길 업무/제출 업무/+ 우리 학교 업무 추가` 버튼 겹침 방지
- 업무분류 딱지를 저채도 파스텔 분류색으로 재정리
- 제출/챙김 행동 배지와 업무분류 딱지 색상 역할 분리
- 직접추가 범례는 회색 계열 유지
- 캐시 방지 버전값 `v245-fix3` 적용

## 유지한 기능
학교 설정, 일정 새로고침, 월 이동, 월 선택 패널, 달력 날짜 클릭 상세, 검색, 완료/미완료, 제출/챙김 필터, 우리 학교 업무 직접 추가 기능은 유지했습니다.
