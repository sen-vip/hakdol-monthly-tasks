# v2.4.1 업무 달력화 업데이트

## 목표
- 추천업무를 길게 나열하는 대신 월별 필수업무 자체를 달력에 얹어 월 흐름을 볼 수 있도록 개선했습니다.
- 달력에서 제출업무와 챙길업무가 몰리는 시기를 한눈에 확인할 수 있게 했습니다.

## 반영 내용
- 월별업무를 시기 기준 대표 날짜에 매핑
  - 월초 → 3일
  - 중순 → 15일
  - 월중 → 16일
  - 20일경 → 20일
  - 하순 → 23일
  - 월말 → 28일
  - 말일까지/말일 → 해당 월 말일
- 달력 칸에 업무명 전체가 아니라 `제출 n`, `챙김 n` 요약 배지 표시
- 날짜 선택 상세 패널에 해당 날짜에 걸어둔 월별업무를 제출업무/챙길업무로 나눠 표시
- 기존 학사일정/시간표/직접추가 일정은 그대로 유지
- 제출업무 색상을 지출 업무딱지와 구분되도록 보라 계열로 변경
- 챙길업무는 민트/초록 계열로 표시
- 기존 추천업무 긴 목록은 날짜 상세에서 제거하고, 월별업무 중심으로 정리

## 유지한 기능
- 업무딱지 컬러
- 학교 설정/학사·시간표 불러오기
- 직접 일정 추가
- 통합달력 기본 펼침
- 월별 업무 목록
- 완료 체크/상세/메모
- 로그인/가입완료 페이지
