# 2026-06-22 추가 수정

## 반영 내용

1. 기본 업무 보기에서 제출업무를 제외하도록 조정
   - `제출`, `신청`, `협조`로 끝나고 안내부서가 있는 업무는 기본 목록에서 제외
   - 필요하면 상단 필터의 `전체`, `제출만 보기`, `제출 제외 보기`로 전환 가능

2. 나이스 연결 보드의 `이번 달 월별업무 연결`에서도 제출업무 제외
   - 공문이 오면 처리하는 제출성 업무는 추천 카드에서 빠지도록 조정
   - 화면 문구도 `이번 달 챙길 업무` 중심으로 변경

3. 우리 학교 일정 직접 추가 기능 추가
   - 나이스 학사일정/시간표에 없는 수련활동, 체험학습 등을 달력에서 직접 추가 가능
   - 날짜를 선택한 뒤 `우리 학교 일정 직접 추가`에 일정명을 입력하면 저장
   - 직접 추가 일정은 `직접추가` 배지로 표시
   - 브라우저 localStorage에 저장되며 서버로 전송하지 않음
   - 직접 추가 일정은 삭제 버튼으로 제거 가능

## 주의

직접 추가한 일정은 현재 브라우저에만 저장됩니다. 다른 PC나 브라우저에서는 다시 입력해야 합니다.
