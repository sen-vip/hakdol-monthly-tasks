# v2.3 학돌 업무판 대개편

## 목표
- 기존 기능은 유지하되 화면을 여러 카드 모음이 아니라 하나의 월간 업무판처럼 재구성
- 신규 행정실 직원이 월별 흐름을 먼저 보고, 업무분장/보기/검색을 한곳에서 조정하는 구조로 변경

## 주요 변경
- 상단 헤더를 업무판 요약형으로 재정리
- 월 선택, 업무분장, 보기 필터, 검색을 하나의 `업무 보기` 패널로 통합
- 월/업무분장 선택 UI를 큰 카드형에서 낮은 칩형으로 변경
- 업무 목록을 월 단위 행 나열에서 시기별 묶음형 업무판으로 변경
  - 월초 / 중순 / 월중 / 월말 / 수시 / 기타
- 업무행 폭 재정리
  - 체크박스 / 시기 / 업무딱지 / 업무명 / 액션 영역 고정
- 카운트 배지가 사라지지 않도록 0 표시 기준 강화
- 학교 일정 확인 영역은 보조 패널 느낌으로 유지
  - 추천업무와 주요 일정 요약 기본 표시
  - 통합달력은 필요할 때 펼침

## 유지
- 업무딱지 컬러
- 전체/챙길 업무/제출 업무 보기
- 검색
- 완료 체크
- 상세 보기
- 메모
- 로그인/가입완료 페이지
- 우리 학교 업무 추가
- 학교 설정/학사일정/시간표/추천업무/통합달력
