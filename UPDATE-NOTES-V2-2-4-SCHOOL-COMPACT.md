# v2.2.4 하단 통합성 정리 업데이트

- 제출업무 필터 용어를 `전체 / 챙길 업무 / 제출 업무`로 정리했습니다.
- 현재 보기 상태에 맞게 목록 제목이 `업무 전체 / 챙길 업무 / 제출 업무`로 표시되도록 보정했습니다.
- 우리 학교 추가 확인 영역을 큰 캘린더 앱처럼 보이지 않도록 compact 보조 카드 형태로 축소했습니다.
- 학교 설정 폼은 기본 접힘으로 변경하고 `학교 설정하기` 버튼으로 열고 닫도록 했습니다.
- 통합달력도 기본 접힘으로 변경하고 `통합달력 보기` 버튼을 눌렀을 때만 표시되도록 했습니다.
- 학교 캘린더 영역에서는 월별업무 중복 목록을 숨기고, 학사일정 기반 추천업무 중심으로 보이도록 정리했습니다.
- 하단 영역 버튼 톤을 상단 필터와 비슷한 연한 회청/블루 계열로 통일했습니다.
