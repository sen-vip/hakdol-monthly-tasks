# v2.3.1 업무행 폭 오류 + 상단 학교 선택 개선

- 업무행 grid 폭 오류 수정
  - 체크박스 / 시기 / 업무딱지 / 업무명 / 액션 칸을 명시적으로 고정
  - 업무명이 좁은 칸에서 세로로 떨어지는 문제 보정
  - 업무명 영역에 min-width: 0, grid-column 고정 적용
- 학교 선택/상태 영역을 상단 업무판 바로 아래로 이동
  - 학교 설정 전: 상단에서 바로 시도교육청/학교명 입력 가능
  - 학교 설정 후: 현재 학교와 교육청/학교코드 표시
  - 상단에서 일정 불러오기/새로고침 접근 가능
- 학교 선택 클릭 수 축소
  - 학교 선택 후 설정 패널을 닫고 학사일정 불러오기를 자동 시작
- 하단 학교 일정 영역 정리
  - 학교 설정 폼 반복 제거
  - 추천업무/주요 일정 요약 + 통합달력 펼침 중심 유지
- 문법 검사
  - app.js 통과
  - neis-extension.js 통과
