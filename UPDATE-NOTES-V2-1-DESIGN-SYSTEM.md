# v2.1 디자인 시스템 정리 업데이트

## 목표
업무딱지 컬러는 유지하면서 상단, 버튼, 필터, 카드, 업무목록의 디자인 언어를 하나로 맞췄습니다.

## 반영 내용
- 전체 컬러 토큰을 남청/블루/회청색 기반으로 재정리
- 앱 헤더, 히어로, 검색/필터 패널, 선택 패널, 월별 업무 카드 스타일 통일
- 버튼 체계를 primary/ghost/filter 성격으로 정돈
- 업무분장 딱지 컬러를 카드 왼쪽 포인트 라인까지 연결
- 업무목록 카드의 제목/설명/메타정보 위계 강화
- 모바일에서 필터와 업무카드가 한 줄 흐름으로 보이도록 보정

## 유지한 것
- 기존 데이터 구조
- 검색/필터/완료 체크/로그인 저장 로직
- 제출성 업무 제외 로직
- signup-complete.html 가입완료 페이지
